# CodeAlpha_Hangman_Game 🎮🐍

Made with Python 🐍 | Beginner Friendly 🚀 | Console Game 🎮**

---

📖 Project Overview
This repository contains a simple **text-based Hangman game** implemented in Python.  
Players attempt to guess a hidden word one letter at a time, with a maximum of **6 incorrect guesses** allowed.  
The project demonstrates **fundamental programming concepts** such as:
- Random word selection
- Loops (`while`)
- Conditional logic (`if-else`)
- Strings and lists
- Modular programming with multiple files

---

📂 Folder Structure
```

## ⚙️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/CodeAlpha_Hangman_Game.git
   ```
2. Navigate into the project folder:
   ```bash
   cd CodeAlpha_Hangman_Game
   ```
3. Run the game:
   ```bash
   python main.py
   ```
